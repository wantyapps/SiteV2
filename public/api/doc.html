The Wantyapps API Documentation

Basics

To log in to the API, add the following headers to the requests:
    	
username: wantyapps
password: password
If you did not use the correct values, then the api will respond with the JSON object below:

        
{
    "success": false,
    "error": "Username or password incorrect"
}
(This error will show up even if you didn't specify any headers.)
If you did use the correct username and password headers, you will see this:
        
{
    "success": true,
    "error": false
}
(Notice that when there are no errors, error will be false.)

Headers

The Action Header

The action header can be used to specify what you want to do after you have logged in to the API.
For example, if we use joke as the value for this header, the API will send us a random joke. Example request:
    	
username: wantyapps
password: password
action: joke
The API will respond with a random joke in the JSON, with a key of joke.
